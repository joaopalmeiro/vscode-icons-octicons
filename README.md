[Octicons](https://github.com/primer/octicons) Visual Studio Code product icon theme.

## Development

- `pnpm install`.
- `pnpm build:ids`.
- `pnpm build`.
- macOS: `Fn`+`F5`.

## Deployment

- `pnpm version minor` or `pnpm version patch` or `pnpm version major`.
- `git tag`.
- `git push --follow-tags`.
